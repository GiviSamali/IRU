Сбор данных для обучения

ИРУ собирает анонимизированные данные о выполненных командах для улучшения системы и обучения собственной модели ИИ. Собираются: текст запроса, тип ОС, выполненные команды и результат (успех/ошибка). Персональные данные не собираются.

Вы можете изменить это решение в любое время.
