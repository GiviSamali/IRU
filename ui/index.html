Пользовательское соглашение

1. Общие положения

ИРУ (Интеллектуальный Режим Управления) — система удалённого управления компьютерами через естественный язык. Используя сервис, вы принимаете условия настоящего соглашения.

2. Ответственность

Пользователь несёт полную ответственность за команды, отправленные на свои устройства. Система выполняет команды от имени пользователя. Разработчик не несёт ответственности за последствия выполнения команд.

3. Безопасность

Токен доступа является ключом к вашим устройствам. Не передавайте его третьим лицам. При компрометации токена немедленно сообщите администратору.

4. Обработка данных

Мы храним: email, имя пользователя, историю команд. Мы НЕ передаём ваши персональные данные третьим лицам. При согласии на сбор данных обучения, команды используются для улучшения ИИ-модели. Вы можете запросить удаление своих данных.

5. Отказ от гарантий

Сервис предоставляется «как есть» (as is). Это бета-версия. Возможны ошибки и перерывы в работе. Разработчик не гарантирует бесперебойную работу.

6. Ограничения

Запрещено использовать систему для: несанкционированного доступа к чужим устройствам, распространения вредоносного ПО, нарушения законодательства РФ.

Сбор данных для обучения

ИРУ собирает анонимизированные данные о выполненных командах для улучшения системы и обучения собственной модели ИИ. Собираются: текст запроса, тип ОС, выполненные команды и результат (успех/ошибка). Персональные данные не собираются.

Вы можете изменить это решение в любое время.
