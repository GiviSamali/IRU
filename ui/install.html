Интеллектуальный Режим Управления

Что такое ИРУ?

ИРУ — система удалённого управления компьютером через естественный язык. Вы описываете задачу текстом, ИИ переводит её в команды и выполняет на вашем ПК.

Что вам понадобится

  • Компьютер на Windows 10/11
  • Токен доступа — получите у администратора
  • Файл agent.zip (скачайте из интерфейса)
  • Файл agent.exe

Шаг 1: Скачайте архив

Нажмите «Скачать агент» в интерфейсе — начнётся загрузка файла agent.zip.

Шаг 2: Распакуйте архив

ВАЖНО: не запускайте agent.exe прямо из архива — он не найдёт нужные библиотеки и не запустится. Распакуйте архив полностью.
  1. Кликните правой кнопкой по agent.zip → «Извлечь всё…» (или используйте 7-Zip / WinRAR).
  2. Укажите постоянную папку, например C:\IRU\. Рекомендуем не внутри Downloads/Temp, чтобы автообновление работало корректно.
  3. В результате в папке должен появиться agent.exe рядом с файлами python3XX.dll и прочими .dll.

Шаг 3: Запустите agent.exe

Двойной клик по agent.exe в распакованной папке. При первом запуске откроется окно ввода токена.

Windows SmartScreen может показать предупреждение — «Подробнее» → «Выполнить в любом случае». Это стандартное поведение для файлов из интернета. Если Защитник Windows или антивирус блокирует — добавьте папку C:\IRU\ в исключения.

Вставьте токен доступа и нажмите «Подключиться». Токен сохранится автоматически — при следующих запусках вводить не нужно.

Вы увидите окно консоли с сообщением о подключении:

[agent] connected

Шаг 4: Войдите в интерфейс

Шаг 1: Скачайте agent.exe

Получите файл agent.exe от администратора. Установка не требуется.

Шаг 2: Запустите agent.exe

Двойной клик по agent.exe. При первом запуске откроется окно ввода токена.

Windows SmartScreen может показать предупреждение при первом запуске. Нажмите «Подробнее», затем «Выполнить в любом случае». Это стандартное поведение для файлов, загруженных из интернета.

Вставьте токен доступа и нажмите «Подключиться». Токен сохранится автоматически — при следующих запусках вводить не нужно.

Вы увидите окно консоли с сообщением о подключении:

[agent] connected

Шаг 3: Войдите в интерфейс

  1. Откройте браузер: irumode.ru
  2. Введите тот же токен доступа
  3. Ваше устройство появится автоматически
  4. Создайте чат и начните общение

Примеры команд

  • Открой браузер
  • Покажи IP-адрес
  • Сколько свободного места на диске?
  • Покажи запущенные процессы
  • Создай файл test.txt на рабочем столе с текстом "привет"
  • Какая версия Windows?
Важно: Агент выполняет команды от имени вашего пользователя Windows. Не запрашивайте удаление системных файлов или форматирование дисков. ИРУ отклонит опасные команды, но будьте аккуратны.

Частые проблемы

Агент не подключается

  • Проверьте интернет-соединение
  • Убедитесь, что токен введён правильно
  • Попробуйте удалить config.json рядом с exe и запустить заново — окно ввода токена появится снова

SmartScreen блокирует запуск

Нажмите «Подробнее» и «Выполнить в любом случае». Файл безопасен, предупреждение связано с отсутствием цифровой подписи.

Устройство не появляется в UI

  • Убедитесь, что агент запущен и показывает [agent] connected
  • Убедитесь, что в UI вы вошли с тем же токеном

Кракозябры в выводе

ИРУ автоматически обрабатывает кодировку (UTF-8), но если проблема остаётся — укажите это в чате, ИРУ попробует другой подход.
