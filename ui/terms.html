1. Общие положения

1.1. Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует порядок использования системы ИРУ (далее — «Сервис»).

1.2. Используя Сервис, вы подтверждаете согласие с условиями настоящего Соглашения. Если вы не согласны, прекратите использование Сервиса.

1.3. Сервис находится на стадии бета-тестирования и предоставляется «как есть» (as is).

2. Описание Сервиса

2.1. ИРУ — система удалённого управления компьютером через естественный язык. Пользователь описывает задачу текстом, ИИ переводит её в команды и выполняет на устройстве.

2.2. Команды выполняются непосредственно на компьютере пользователя через программу-агент, установленную на устройстве.

3. Права и обязанности пользователя

3.1. Пользователь обязуется не использовать Сервис для:

  • несанкционированного доступа к чужим устройствам или данным;

  • попыток обхода систем защиты Сервиса;
  • создания чрезмерной нагрузки на сервер (флуд, DDoS).

3.2. Пользователь несёт полную ответственность за команды, отправляемые через Сервис, и их последствия на своих устройствах.

3.3. Токен доступа является персональным. Передача токена третьим лицам запрещена.

4. Сбор данных

4.1. Сервис может собирать анонимизированные данные о взаимодействии с системой (текст запросов, выполненные команды, информация об ОС) исключительно для улучшения качества Сервиса и обучения модели.

4.2. Сбор данных осуществляется только при явном согласии пользователя (переключатель в интерфейсе). Пользователь может отозвать согласие в любой момент.

4.3. Персональные данные (имя пользователя, содержимое файлов) не передаются третьим лицам и не продаются.
